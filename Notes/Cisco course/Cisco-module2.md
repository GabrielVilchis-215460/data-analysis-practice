## Observaciones, Variables y Valores
Variable -> Cualquier cosa que varía de una instancia a otra, que se puede medir y cuyo valor se puede manipular o controlar en escenarios teóricos.

Punto de datos -> Es el valor o conjunto de valores de una observación específica. (Cada observación puede considerarse y representarse como un registro en un BD o una fila en una hoja de calculo de Excel.)

La colección de observaciones constituye el conjunto de datos para su análisis. Y estas mismas, suelen tener un propósito, y las variables incluidas dependerán de su relevancia para ese propósito.

## Tipos de variables
# Variable Categórica
Las variables categóricas indican membresía a un grupo determinado y tienen un valor cualitativo discreto o especifico. Ademas, se clasifican en 2 tipos:
1. Nominales ->  Son variables que constan de dos o más categorías discretas cuyo valor se asigna en función de la identidad del objeto. Algunos ejemplos son sexo, color de ojos o tipo de animal.

2. Ordinales -> Estas son variables compuestas por dos o más categorías en las que el orden es importante en el valor. Algunos ejemplos son el rango de clases de los estudiantes (1ro, 2do, 3ro) o las escalas de las encuestas de satisfacción (insatisfecho, neutro, satisfecho).

# Variable Numérica
Las variables numéricas son valores cuantitativos:
1. Continuas -> Estas son variables cuantitativas y pueden medirse a lo largo de una secuencia o un rango de valores. Hay dos tipos de variables continuas: las variables de intervalo pueden tener cualquier valor dentro del rango de valores, por ejemplo, la temperatura o el tiempo; Las variables de relación son variables de intervalo especiales donde un valor de cero (0) puede significar que no hay nada de esa variable, por ejemplo, los ingresos o el volumen de ventas.

2. Discretas -> Estos tipos de variables continuas son cuantitativos pero tienen un valor específico de un conjunto de valores finito. Los ejemplos incluyen el número de sensores habilitados en una red, o el número de automóviles en un estacionamiento.
